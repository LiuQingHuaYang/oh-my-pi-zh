# Changelog

## 2.0.0 (2026-06-21)

### Breaking Changes

- **不再作为 npm 包发布**。omp 从 v16 开始改用编译二进制分发，旧的 npm 补丁方式已失效。
- 移除了 `index.js`、`data.js`、`patch.js`、`unpatch.js`、`package.json`。

### Added

- `patch_omp_zh_v2.py`：全新的 Python 补丁脚本，翻译数据集中管理，按文件+类别组织。
- `.github/workflows/build.yml`：GitHub Actions CI，自动编译中文版二进制。
- 提供预编译二进制下载（Releases）。

### Changed

- 翻译数量从 ~500 处提升至 **~950+ 处**。
- 新增翻译：工具/Shell/任务/提供商栏目下的所有设置项。
- 新增翻译：斜杠命令菜单中全部命令和子命令描述。
- 新增翻译：调试菜单全部项。
- 新增翻译：键盘快捷键全部描述。
- 翻译策略：分组标题、标签纯中文；技术术语双语标注。

## 1.0.0 (2026-06-04)

- 初始版本
- npm 包形式分发
- 翻译范围：设置界面、斜杠命令、工具摘要、调试菜单、欢迎界面、键盘绑定等
- 总计约 500+ 处翻译
- 双语格式：保留原文 + 中文标注
